# StarCycle Mandrill Email Templates

Email templates using [MJML](https://mjml.io/download)

### Build
`npm run build`

### Development
To output HTML:
`mjml -r index.mjml -o index.html`

To watch changes:
`mjml --watch index.mjml -o index.html`
